## Getting Started

To connect Muse 2 and see live data collection run:
```sh
python ./main_menu.py
```

To start the video game run:
```sh
python ./MyGame.py
```

To play the video game with concentration detection, leave the **main_menu.py** running, then run **MyGame.py**.
